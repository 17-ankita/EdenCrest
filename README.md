# EdenCrest — Luxury Villa Living

A modern luxury real-estate landing page designed for showcasing premium villas, elegant living spaces, and high-end community amenities.
Built with a clean UI, smooth animations, responsive layouts, and immersive visuals to deliver a premium user experience.

## Live Demo

https://eden-crest.vercel.app/


# Project Overview

EdenCrest is a responsive real-estate website inspired by modern luxury township and villa projects.
The website focuses on elegant presentation, smooth navigation, immersive property showcases, and visually rich sections for premium user engagement.

The project was created to practice and demonstrate:

* Modern frontend development
* Responsive website design
* Real-estate landing page UI/UX
* Interactive animations
* Clean component structuring
* Professional portfolio-ready design

# Tech Stack

* HTML5
* CSS3
* JavaScript
* AOS (Animate On Scroll)
